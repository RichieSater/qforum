I passed str(sid) into the function below to get the tuples. These are all from 80-100 in the dollar vol universe, mostly > 90. 2010 is as far as I went back. There's probably more because I had to add the if statement to avoid IndexErrors. 

def id_tuple(sid):  
    tup = [i.strip("[])") for i in sid.split("(")[-1].split()]  
    if len(tup) == 2:  
        return int(tup[0]), tup[1]
