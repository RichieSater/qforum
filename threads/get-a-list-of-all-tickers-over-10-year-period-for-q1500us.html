Hi All,

I'm looking to obtain a list of all tickers over a 10 year period in Q1500US() (I'm assuming it doesn't remain static -- someone tell me if I'm wrong). However, when I run my code, I am running into an issue where it says "WARN Logging limit exceeded; some messages discarded".

My code is very basic. I am setting my universe to Q1500US() and running this in pipeline:
context.output = pipeline_output('pipeline')
for equity in context.output.index:
print equity

It seems to work for the first few equities, but it seems that there's a limit to how much I can log to the output console: https://www.quantopian.com/posts/warn-logging-limit-exceeded-some-messages-discarded. Can anyone assist?

Limit content sent to the logging window at around 1000 bytes at a time to stay under one of the logging limits.
That will address one part of your goal.
QTradableStocksUS is the one we need to be working with toward the contest and fund so that's what I used for this example.

        if len(syms) > 1000:  

from quantopian.pipeline  import Pipeline  
from quantopian.algorithm import attach_pipeline, pipeline_output  
from quantopian.pipeline.filters import QTradableStocksUS

def initialize(context):  
    context.done = 0  
    attach_pipeline(Pipeline(screen=QTradableStocksUS()), 'z')

def before_trading_start(context, data):  
    if context.done: return    # just one day for this example

    out = pipeline_output('z')

    syms = ''  
    symbol_names = [s.symbol for s in out.index]  
    for sym in sorted(symbol_names):  
        syms += sym + ' '  
        if len(syms) > 1000:  
            log.info(syms)  
            syms = ''

    context.done = 1
